# 🛍️ Customer Shopping Behavior Analysis
## 📖 Project Overview

This project analyzes customer shopping behavior using a retail dataset containing 3,900 customer transactions and 16 variables. The objective is to identify purchasing patterns, customer preferences, spending behavior, and factors that influence buying decisions.

The project demonstrates the complete data analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL-based business analysis, and interactive dashboard development using Power BI.

## 🎯 Objectives
* Analyze customer purchasing behavior.
* Identify revenue-generating product categories.
* Explore customer demographics and spending patterns.
* Evaluate the impact of discounts and subscription programs.
* Create interactive dashboards for business decision-making.
## 🛠️ Tools & Technologies

 🗄️ MySQL
 
 📊 Power BI

## 📂 Dataset Information
📌 Records: 3,900 customer transactions

 📌 Features: 16 variables
 
 📌 Source: Kaggle Customer Shopping Behaviour Analysis Dataset
## 🔍 Key Findings
👕 Clothing generated the highest revenue among all product categories.

👥 Young Adult customers contributed the highest overall revenue.

🎟️ Discounts and subscription programs showed limited impact on customer spending.

📈 Customer purchasing behavior remained relatively consistent across demographic groups.
